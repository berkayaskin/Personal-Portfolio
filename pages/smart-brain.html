Smart Brain

Smart Brain is a web application created with React. It uses a machine learning API called Clarifai to detect faces in images submitted via an input form. It utilizes a database to store users and ranks them according to the amount of images submitted for face detection.

Features
  • Adding Images to Detect Faces
  • User Profiles with Ranks
  • User Authentication
Technologies

HTML5, CSS3, Tachyons, Javascript, ReactJS, Clarifai API, NodeJS, Bcrypt.js, ExpressJS, PostgreSQL
